# Dashboard de Vendas Xbox no Excel

## Descrição do Projeto
Este projeto foi desenvolvido como parte de um desafio de bootcamp com o objetivo de criar um **dashboard de vendas no Excel**, transformando dados brutos em informações visuais claras e úteis para análise de desempenho.

O foco do dashboard é facilitar a leitura dos principais indicadores de vendas e apoiar a tomada de decisão a partir de dados organizados, gráficos e métricas resumidas.

---

## Objetivo
Construir um dashboard de vendas que permita visualizar, de forma rápida e intuitiva:

- faturamento total;
- quantidade de assinaturas vendidas;
- desempenho por tipo de assinatura;
- comportamento das vendas ao longo do tempo;
- participação de produtos complementares na receita.


## Estrutura do Dashboard
O dashboard foi organizado para destacar as informações mais relevantes da base de vendas, seguindo o modelo proposto no material de apoio, mas com um diferencial analítico adicional.

### Indicadores principais
- **Faturamento Total**
- **Total de Assinaturas EA Play**
- **Total de Assinaturas Minecraft Season Pass**

### Gráficos e visualizações
- **Faturamento por ano**
- **Faturamento por tipo de assinatura**
- **Distribuição das vendas por renovação automática**

### Diferencial aplicado
Além do modelo base, foi incluído o indicador:

- **Share de Add-ons na Receita Total**

Esse indicador mostra quanto da receita total é representado por produtos complementares, agregando uma visão mais estratégica ao dashboard e trazendo um nível extra de análise para a entrega.

## Como Reproduzir o Projeto
1. Baixe os arquivos deste repositório;
2. Abra o arquivo `dashboard_xbox_kaw_diferencial.xlsx` no Microsoft Excel;
3. Verifique a aba com a base de dados;
4. Analise os indicadores e gráficos no dashboard final.

Caso deseje editar ou atualizar o projeto:
1. Substitua ou atualize os dados na base;
2. Revise as fórmulas e tabelas auxiliares;
3. Atualize os gráficos, se necessário.
